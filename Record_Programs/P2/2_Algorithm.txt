1. Start the program and include stdio.h.
2. Define `push()` to check overflow, take input, increment `top`, and insert the value.
3. Define `pop()` to check underflow, print the removed value, and decrement `top`.
4. Define `display()` to print all stack elements from `top` to 0, or show “Stack is empty.”
5. In `main()`, declare `stack[100]`, `top = -1`, `n`, and `choice`.
6. Read the stack size `n`.
7. Loop a menu with options: Push, Pop, Display, Exit.
8. Use a switch: call `push()`, `pop()`, `display()`, or exit the loop.
9. Repeat until Exit is chosen.
10. End program.
